    Deskripsi Web

    Auto-EDA Dashboard

    Aplikasi DS Generator adalah platform web interaktif untuk analisis data eksploratif (Exploratory Data Analysis) secara otomatis. Dibangun menggunakan Flask (Python) sebagai backend, Jinja2 sebagai template engine, dan Plotly.js untuk visualisasi chart yang dinamis. Fitur utama meliputi: upload dataset (CSV/XLSX), data cleaning otomatis, ringkasan statistik deskriptif, visualisasi interaktif, deteksi time series, hingga generation insight berbasis AI. Dashboard ini dirancang agar pengguna tanpa latar belakang statistik kuat pun dapat memperoleh insight dari data secara cepat dan akurat.
